# Security Notes

## Keep out of this repo

- real tokens
- live tunnel URLs
- OAuth credentials
- raw OpenClaw transcripts unless redacted
- Cloudflare quick tunnel URLs
- destructive helper scripts not reviewed
- machine scheduler state
- private QR codes or private phone dashboard URLs

## Allowed in this repo

- redacted census JSON
- seed rosters
- schemas
- local app code
- example manifests
- hook templates
- redacted config examples
- field reports
- artifacts that do not contain secrets
- milestone logs

## MVP posture

Read-only only. Expedition HQ should observe first, not control.
