# Gamification Rules

## Reward real progress only

XP should be earned when:
- an expedition meaningfully advances
- an artifact is created
- a blocker is resolved
- a field report preserves useful history
- a milestone is reached

Do not award XP for:
- random pings
- opening files
- fake activity
- empty summaries
- repeated no-op status checks

## Starter badges

- Map Room Founded
- First Roster Rendered
- First Field Report
- First Little Guys
- Checkpoint Built
- Archive Rules Drafted
- Dormant Engineer Awakened
