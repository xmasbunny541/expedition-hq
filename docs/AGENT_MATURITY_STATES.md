# Agent and System Maturity States

| State | Meaning |
|---|---|
| awake | Active and currently useful |
| on_call | Semi-active or available for targeted work |
| dormant | Configured but not observed active |
| test_mode | Working only in a test/local route |
| temporary_route | Active but not safe/stable long term |
| fragile_link | Working but route-sensitive |
| sleeping | Candidate inactive |
| blueprint | Planned but not built |
| blocked | Known blocker prevents production use |
| stable | Reliable host/system |
| unknown | Not enough evidence |

Use these states for visual behavior and labels.
