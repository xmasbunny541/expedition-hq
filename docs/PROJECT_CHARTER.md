# Project Charter: Expedition HQ

## Purpose

Expedition HQ is a local-first progress observatory for August's AI ecosystem.

It exists to:
- make AI agents and agent-like systems visually identifiable
- answer "what are the little AI guys doing?"
- track multiple concurrent projects as expeditions
- preserve proof of progression through field reports and artifacts
- add light gamification for motivation
- stay safe by starting read-only

## North Star

A living, readable dashboard where the visual layer is fun, but the underlying records stay practical and trustworthy.

## MVP focus

- Read-only specialist roster
- Expedition board
- Event ledger
- Field report timeline
- Milestone gallery
- Seed-driven Bureau HQ view
